# IDKick

Live page: [mysticalg.github.io/IDKick](https://mysticalg.github.io/IDKick/)

Legacy VST plugin binary package for `IDKick.dll`.

## Contents

- `IDKick.dll`

## Install

Copy `IDKick.dll` into your VST plugins folder and rescan plugins in your host.

## Notes

This repository currently contains the packaged binary rather than the original source project.
